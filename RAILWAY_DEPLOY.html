Railway Deployment Guide

1. Create Railway Account

Go to railway.app and sign up with GitHub

2. Deploy Database

Click "New Project" → "Provision PostgreSQL"
Note the connection details from Variables tab

3. Deploy Web App

Click "New Project" → "Deploy from GitHub repo"
Select this repository
Railway will auto-detect the Dockerfile

4. Set Environment Variables

DATABASE_URL=postgresql://username:password@host:port/database
JWT_SECRET_KEY=your-super-secret-jwt-key-here-minimum-32-characters
ASPNETCORE_ENVIRONMENT=Production

5. Initialize Database

Connect to PostgreSQL using Railway's built-in Query tool
Run the script from Scripts/railway-init.sql

6. Access Your App

Railway will provide a public URL
Login with: admin / admin123

Free Tier Limits

  • 500 hours/month execution time

Troubleshooting

  • Check logs in Railway dashboard
  • Ensure DATABASE_URL format is correct
  • Verify all environment variables are set
