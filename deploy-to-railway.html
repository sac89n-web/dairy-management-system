Step 1: Push Code to GitHub

  2. Push your code:
    git init git add . git commit -m "Initial commit - Dairy Management System" git branch -M main git remote add origin https://github.com/yourusername/dairy-management.git git push -u origin main

Step 2: Deploy Web Application

  1. Go to railway.app
  2. Click "New Project"
  3. Select "Deploy from GitHub repo"
  4. Choose your dairy-management repository
  5. Railway will auto-detect the Dockerfile and deploy
